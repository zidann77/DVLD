# 🚦 DVLDProject – Driving Vehicle License Department System

A **desktop application** built with **WinForms, SQL Server, and 3-Tier Architecture**, designed to streamline government vehicle licensing operations.  
Features include full license services (new, renew, replace, retake tests), test appointment management, license detainment & release, user management and real-time data refresh—showcasing hands-on experience in desktop app development, database design, and 3-tier architecture implementation.

---

## 🚀 Features

- 📄 Full license services: new, renew, replace, retake tests
- 📍 Support Local License Classes
- 🌍 International License Issue
- 📅 Test appointment management
- 📝 Tests Managment (Written, Vision and Street)
- 💸 License detainment & release  
- 👤 People, User and Drivers management 
- 🔄 Real-time data refresh  
- 🏗 Clean 3-tier architecture (UI / BLL / DAL)
